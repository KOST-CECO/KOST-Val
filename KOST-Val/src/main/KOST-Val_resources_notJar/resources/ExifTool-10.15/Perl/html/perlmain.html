ActivePerl User Guide

Welcome to ActivePerl

Thank you for using ActivePerl, ActiveState's quality-assured binary build of Perl, available for Linux, Solaris and Windows. As part of ActiveState's support for Perl, ActiveState provides the ActivePerl binary packages free to the community. ActivePerl includes:

  • Perl, the binary core distribution
  • The Perl Package Manager , for installing Perl extension modules
  • Complete documentation

The Windows version of ActivePerl also includes:

  • Perl for ISAPI, an IIS plug-in that makes perl CGI faster
  • PerlScript, an ActiveX scripting engine, like JavaScript or VBScript with a Perl brain
  • PerlEz, for embedding Perl the easy way

Refer to the Release Notes for information about this version of ActivePerl.
